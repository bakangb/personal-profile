 Personal Profile Website

A responsive **personal portfolio webpage** built with **HTML5**, **Bootstrap 5**, and custom CSS.  
It showcases personal details, skills, education, and contact information.

---

 Project Structure
`index.html` → Main webpage structure  
 `style.css` → Custom styles  
 `images/profile.jpg` → Profile picture  


 Features
 Responsive Navbar with section links  
 Hero Section with profile image and tagline  
About Section with description  
Skills Section listing technical abilities  
 Education Section with academic background  
 Contact Section with email info  
 Footer with copyright  


Technologies Used
 HTML5  
 Bootstrap 5  
 CSS (style.css)  
 JavaScript (Bootstrap Bundle)  


 How to Run
 Download or clone the project files.  
 Open `index.html` in a browser.  
 Ensure `style.css` and `images/profile.jpg` are in the correct paths.
